hexo搭建记录

hexo 搭建博客
环境 kali2.0
https://nodejs.org/en/ 从node.js下载tar.xz文件
xz -d name 解压为tar文件
tar -xvf name 解压tar
cd node…/bin
ln -s ～/xxx/xxx/node /usr/local/bin/node
ln -s ~/xxx/xxx/npm /usr/local/bin/node
npm install -g hexo
mkdir blog;cd blog
hexo init 如果无hexo命令 回到设置全局域名的地方，应该有个hexo，将hexo添加到全局域名。
hexo generate
hexo server
http://localhost:4000 应该能成功，打开这个网址

vim _config.yml #最下面改为
deploy:
type: git
repository: https://github.com/ranyand/ranyand.github.io.git
branch: master
npm install hexo-deployer-git –save
hexo deploy
此时输入 http://ranyand.github.io 就有网站了。
常用命令
hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #将.deploy目录部署到GitHub
hexo help #查看帮助
hexo version #查看Hexo的版本
